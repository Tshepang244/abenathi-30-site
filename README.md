# Abenathi Turns 30 

This package is ready for you to drop in your images and deploy to Vercel.

## Where to put your images
- **Hero (Home):** `assets/img/hero.jpg`
- **Gallery:** `assets/img/gallery1.jpg` … `assets/img/gallery12.jpg`


## Links already wired
- Video teaser (Google Drive preview iframe)
- RSVP button (Google Form)
- Venue (Google Maps link)

## Deploy (Vercel)
1. Go to Vercel Dashboard → **+ Add Project** → **Upload**.
2. Select the folder `abenathi_30_site_IMAGES_READY`.
3. Click **Deploy**. Optionally rename your domain (e.g. `abenathi30.vercel.app`).
